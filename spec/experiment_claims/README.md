# 實驗主張能力索引

本目錄以 **task** 為單位整理「這個 benchmark task 能回答哪些研究問題，以及閱讀結果時該看哪些 scope 與 metrics」。

這一層的用途是：

- 幫文件作者快速定位哪個 task 可以回答哪個 question
- 記錄每個 question 需要的 variant / comparison / prompt scope

這一層不包含：

- run-specific verdict
- 類似 verdict label 的狀態欄位
- 分數、分子分母或例證數字
- 可直接引用到論文的結果結論
